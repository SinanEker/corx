About:
------

Corx is a mini php framework, built for performance and better code handling.

Required:
    php 5.4.16 with following configurations:
    - mysqli enabled
    - zlib enabled
    - zip enabled
    - SPL enabled

To use corx just include Init.php in your main file. 

Features:
---------
* Gzip
* Http wrapper
* Plugins avaiable 
* Mysqli plugin (buggy)
